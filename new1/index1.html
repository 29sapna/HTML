Frontend Development

CSS JavaScript

Front-end web development is the development of the graphical user interface a website through the use of HTML, CSS and JavaScript so users can view and interact with that website.

  • HTML: is the standard markup language for Web pages. With HTML you can create your own Website. HTML is easy to learn - You will enjoy it!

  • CSS: stands for Cascading Style Sheets CSS describes how HTML elements are to be displayed on screen, paper, or in other media.

    1. Pure CSS

    2. Bootstrap (Framework)

    3. Tailwind (Framework)

    4. Sass (Preprocessor)

  • JavaScript: is the most popular scripting language for the Web. It is easy to learn, lightweight, cross-platform, single-threaded, and interpreted compiled language.

    1. Vanilla JS

    2. React (Framework)

    3. Angular (Framework)

    4. Typescript (Preprocessor)
